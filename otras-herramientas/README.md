# 🧰 Otras Herramientas

Colección de scripts y herramientas independientes, cada una en su propia carpeta.

| Carpeta | Qué hace | Stack |
|---|---|---|
| [`ml-scrapper/`](./ml-scrapper) | Tracker de precios de Mercado Libre Chile — scraping + dashboard con alertas de descuentos | Python, Flask, React, Chart.js, SQLite |
| [`image-blaster/`](./image-blaster) | Procesamiento batch de imágenes (bordes, blanco/negro, thumbnails) | Node.js |
| [`tarot-conversion-tools/`](./tarot-conversion-tools) | Scripts de conversión de cartas de tarot a paint-by-numbers | Node.js + Python |
| [`transcripts/`](./transcripts) | Descargador de historial y transcripts de YouTube | Python |
| [`utilidades/`](./utilidades) | Scripts sueltos de uso personal | Python |

Cada carpeta es independiente — revisa su propio README (si existe) para instrucciones específicas.
