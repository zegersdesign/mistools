# Changelog

## [1.0.0] — 2026-06-24

### Added
- ClaudeCleaner — temp, cache, prefetch, papelera with 10 configurable zones
- Claude Antimalware — Windows Defender Quick/Full/Custom scan
- Disk Analyzer — per-drive folder size ranking with progress bar
- Startup Manager — user/system filter, reads registry + WMI + startup folders
- System Info — CPU, RAM, GPU, disks, network, top processes with section selector
- Real-time SSE streaming from PowerShell to browser terminal
- Options modal for every tool
- Expandable detail section on each card
- Dark theme UI with sidebar navigation
- `/drives` endpoint for dynamic drive detection
- `iniciar.bat` launcher with auto Flask install
- Scheduled task support for daily ClaudeCleaner run
