MisTools

Tu caja de herramientas local para Windows — desde el navegador.

Windows 10/11 Flask + PowerShell + Node 100% local · sin nube Código abierto
Ver en GitHub Cómo instalar

Herramientas incluidas

ClaudeCleaner

Limpia temporales, cache de browsers, prefetch y papelera. Libera GB en segundos.

Claude Antimalware

Escaneo rápido, completo o de carpeta con Windows Defender integrado.

Disk Analyzer

Ranking de las carpetas más pesadas de cualquier disco, con barras visuales.

Startup Manager

Lista todo lo que arranca con Windows: registro, WMI y carpetas de inicio.

System Info

CPU, RAM, GPU, discos, red, IP pública y top de procesos de un vistazo.

Wallpaper Downloader NUEVO

Pega la URL de cualquier galería y descarga todos los wallpapers de alta resolución.

Instalación

  1. Clona o descarga el repositorio.
  2. Instala dependencias: Python 3.11+ y Node.js (para el Wallpaper Downloader).
  3. Doble click en iniciar.bat — abre localhost:7777 en tu navegador.
# Clonar
git clone https://github.com/zegersdesign/mistools.git
cd mistools

# Instalar dependencias de Python
pip install -r requirements.txt

# Arrancar (o doble click en iniciar.bat)
python app.py

Importante

Esta página es la vitrina del proyecto. MisTools corre en tu propia máquina: las herramientas ejecutan comandos de Windows (limpieza, Defender, análisis de disco) que solo tienen sentido localmente y por seguridad no pueden ni deben correr en un servidor en la nube. Por eso el flujo es: descargar el código desde aquí → ejecutarlo en tu PC.
