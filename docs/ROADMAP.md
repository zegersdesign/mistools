# Roadmap

## v1.0 — Current
- [x] ClaudeCleaner with per-category checkboxes
- [x] Claude Antimalware (Windows Defender integration)
- [x] Disk Analyzer with drive selector
- [x] Startup Manager with user/system filter
- [x] System Info with section selector
- [x] Real-time streaming output (SSE)
- [x] Dark UI with sidebar, modals, terminal panel
- [x] Detail section per tool card
- [x] Daily scheduled task for ClaudeCleaner

## v1.1 — Next
- [ ] Process Manager — list and kill processes by name or PID
- [ ] Network Monitor — live upload/download speed per adapter
- [ ] Port Scanner — see what's listening on which ports
- [ ] Scheduled Tasks Manager — view/enable/disable Windows Task Scheduler entries
- [ ] Persist user preferences (last used options per tool) via localStorage

## v1.2
- [ ] Windows Services Manager — start/stop/disable services
- [ ] Hosts File Editor — add/remove entries with validation
- [ ] Duplicate File Finder — find duplicates by hash in a folder
- [ ] System Restore Points — list and create restore points

- [ ] Compile to single .exe with PyInstaller (zero-dependency install)
- [ ] System tray icon — run in background, access from tray
- [ ] Auto-updater — pull latest scripts from GitHub

- [ ] Light mode toggle
- [ ] Export reports as PDF/HTML
